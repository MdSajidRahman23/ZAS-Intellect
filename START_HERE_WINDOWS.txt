ZAS-Intellect Adaptive Viva Demo - Windows Quick Start

1) Extract the ZIP here:
   D:\ZAS-Intellect

2) Open PowerShell and run:

   cd "D:\ZAS-Intellect\zas_intellect_final_python"
   Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
   .\scripts\setup_windows.ps1
   .\scripts\reset_db.ps1
   .\scripts\start_windows.ps1

3) Open browser:
   http://127.0.0.1:8000

5) Adaptive viva behavior:
   The first question starts at Standard level.
   If the student answers well, the next question becomes Advanced.
   If the student answers poorly, the next question becomes Foundation level.
   Official marks are difficulty-aware: easier questions are capped lower, advanced questions can receive a small bonus.

6) Secure proctoring:
   Viva timer and question access start only after Start Secure Viva passes camera + microphone + recording + full-screen checks.
   Camera and microphone are mandatory. Without both, the viva cannot start.
   Full-screen exit / ESC / tab hide / focus loss ends the viva and flags Security Violation.
   Video evidence is saved under app\data\recordings.

7) Grok/xAI:
   This build uses AI_PROVIDER=auto.
   If XAI_API_KEY is empty, the system automatically uses offline fallback.
   To enable Grok, open .env and set:
   XAI_API_KEY=your_xai_api_key_here
   XAI_MODEL=grok-4.3

8) If anything breaks, reset database:
   .\scripts\reset_db.ps1
