# ZAS-Intellect Demo Evidence

ZAS-Intellect is an AI-driven adaptive viva and academic integrity layer for university assignment assessment.

## Demo Mode

For stable competition demonstration, the project can run in Offline Fallback mode.

External AI providers are supported:

- Grok / xAI
- Gemini
- Offline fallback

If Grok/Gemini API credits, quota, or network fail, the system safely falls back to offline question generation.

## Evidence Included

This repository includes demo evidence files under:

- `app/data/uploads/`
- `app/data/recordings/`

The recording folder contains webcam evidence chunks from viva sessions.

## Proctoring Features

- Camera check
- Microphone check
- Secure full-screen start
- Video recording chunks
- Motion / stillness monitoring
- MediaPipe face proctoring initialization
- Teacher evidence review support

## Important Note

ZAS-Intellect does not automatically punish students. It provides evidence-backed decision support. Final academic decision remains with the teacher.
